# Kế Toán Sen Vàng - Website

Trang web chính thức của công ty dịch vụ kế toán chuyên nghiệp Kế Toán Sen Vàng, cung cấp các dịch vụ kế toán, thuế và hỗ trợ doanh nghiệp tại Việt Nam.

## Mô tả

Website Kế Toán Sen Vàng là một ứng dụng web hiện đại được xây dựng với React và Vite, cung cấp thông tin về các dịch vụ kế toán, thuế, và hỗ trợ doanh nghiệp. Trang web có giao diện thân thiện, responsive và dễ sử dụng trên cả thiết bị di động và máy tính.

## Tính năng chính

- **Responsive Design**: Giao diện phù hợp với mọi thiết bị (mobile, tablet, desktop)
- **Navigation Đa Ngôn Ngữ**: Menu điều hướng dễ sử dụng với 5 trang chính
- **Hiển thị dịch vụ chuyên nghiệp**: Trang dịch vụ chi tiết, rõ ràng
- **Hiệu ứng động mượt mà**: Hiệu ứng animate và counter cho trải nghiệm người dùng tốt hơn
- **Liên hệ dễ dàng**: Form liên hệ và các nút liên lạc nổi bật
- **Tối ưu hiệu suất**: Sử dụng React Router, Tailwind CSS và các thư viện hiện đại

## Cấu trúc trang web

- **Trang chủ** (`/`): Giới thiệu giá trị cốt lõi, lợi ích dịch vụ và thống kê ấn tượng
- **Dịch Vụ** (`/dich-vu`): Các dịch vụ kế toán, thuế chuyên nghiệp
- **Kho & Cung Ứng** (`/dich-vu-khac`): Các dịch vụ hỗ trợ doanh nghiệp khác
- **Giới Thiệu** (`/gioi-thieu`): Giới thiệu về công ty và đội ngũ chuyên gia
- **Liên Hệ** (`/lien-he`): Thông tin liên hệ và form gửi yêu cầu

## Công nghệ sử dụng

- **React 19**: Thư viện JavaScript cho xây dựng giao diện người dùng
- **React Router DOM**: Quản lý điều hướng và routing
- **Tailwind CSS**: Framework CSS hiện đại cho styling nhanh chóng
- **Vite**: Công cụ build nhanh chóng và hiệu quả cho dự án React
- **EmailJS**: Gửi email liên hệ trực tiếp từ frontend
- **Lucide React**: Icon hệ thống đẹp và hiện đại
- **React Hook Form**: Quản lý form hiệu quả và dễ sử dụng

## Cài đặt

1. Clone repository này về máy tính của bạn:
   ```bash
   git clone [URL_REPOSITORY]
   cd websenvang
   ```

2. Cài đặt các package cần thiết:
   ```bash
   npm install
   ```

3. Chạy ứng dụng ở môi trường phát triển:
   ```bash
   npm run dev
   ```

4. Mở trình duyệt và truy cập: `http://localhost:5173`

## Triển khai sản phẩm

1. Tạo bản build sản phẩm:
   ```bash
   npm run build
   ```

2. Chạy bản preview để kiểm tra:
   ```bash
   npm run preview
   ```
